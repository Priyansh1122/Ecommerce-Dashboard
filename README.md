# Ecommerce-Dashboard

1-Project Overview-

This project focuses on analyzing sales, profit, and quantity across different years, regions, and product categories using an interactive Power BI dashboard.

2-Objective-

The main objective is to enable dynamic comparison of current year vs previous year performance using slicers, along with detailed insights across regions and categories.

3-Dataset Information-

The dataset used in this project is sourced from Kaggle.

4-Data Cleaning-

Data preprocessing was performed using Power Query, including:

Removal of null values
Elimination of duplicate records

5-Key Features-
            Interactive dashboard with multiple visualizations:
            Bar charts
            Line graphs
            Pie & donut charts
            Waterfall chart
            Use of slicers for dynamic filtering (year, region, category)


6-DAX Calculations-    Key measures created using DAX include: YTD Sales
                                                               YTD Profit
                                                               YTD Profit Margin
                                                               Average Order Value
                                                               Average Profit

7-Insights & Findings:
                     Sales peaked in 2021
                     The West region has the highest number of customers


8-How to Use-
            Download the .pbix file
            Open it in Microsoft Power BI Desktop
